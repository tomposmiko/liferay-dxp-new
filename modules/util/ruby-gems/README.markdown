# Ruby Gems

The Ruby Gems module provides a set of Ruby gems that Liferay can access when
using JRuby. The gems included with this module are listed below:

- `chunky_png-1.3.8`
- `compass-1.0.1`
- `compass-core-1.0.3`
- `compass-import-once-1.0.5`
- `ffi-1.9.18-java`
- `multi_json-1.12.2`
- `rb-fsevent-0.10.2`
- `rb-inotify-0.9.10`
- `sass-3.4.25`
